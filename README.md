# markdown-cheat-sheet
A Markdown cheat sheet providing a quick overview of all the Markdown syntax elements.

## Basic elements
| **Element** | **Syntax** |
| ----------- | ----------- |
| Heading | `# H1`  `## H2`  `### H3`|
| Link | `[text](https://example.com)` |
| Image | `![alt text](image.png)` |

## Text elements
| **Element** | **Syntax** |
| ----------- | ----------- |
| Bold | `**Bold text**` |
| Italic | `*hi*` |
| Blockquote | `> blockquote` |
